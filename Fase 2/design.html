Proposito del sitio

Este sitio se hace con el fin de dar solución el taller de la fase 2 de diseño de sitios Web, donde estaremos tratando los temas del estudiante 4
(Flex Box, ulstas y enlaces), como usar cada una de estas herramientas y que afectan a la hora de diseñar el sitio.

Flex box

El flexbox o también llamado posicionamiento flexible, ayuda a acomodar de una mejor manera los elementos de una página, tanto en dimensión como en
orientación, sin importar la estructura que ya tenga el modelo HTML aun no es 100% compatible con todos los navegadores, la idea principal
de un contenedor flexible es que se pueda adaptar al área disponible, teniendo esto en cuenta, también habrían unos flex ítems que se acomodarían dentro del flex
container, en este caso no hay posicionamiento de izquierda o derecha , si no que tendríamos (main axis) y (cross axis) que se ajustarían a los ejes que dispone el dispositivo

(Recio García, 2016)

Listas

En HTML5 existen tres tipos de ulstas
ol= ulstas numéricas
ul= ulstas con símbolos o viñetas
dl= Se usan generalmente para definir un término

(Recio García, 2016)

Enlaces

Los hipervínculos nos permiten movernos a diferentes partes del documento o a otros documentos externos es decir que son varios documentos enlazados
que pueden ayudar a complementar información
Para poner un enlace se usa la etiqueta "a"

(Recio García, 2016)
